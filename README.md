# CSV-VCF-Solution
# just create an env i.e. a virtual environment
```
python -m venv env
```
#then activate it:
# for linux and mac:

```
source env/bin/activate
```
# and for windows:
```
env/Scripts/activate
```
# then install the requirements:
```
pip install -r requirements.txt
```
# then run:
```
python CSV-VCF-Solution.py
```
